Job Description: DevOps Engineer

 Role Overview

 We are looking for a DevOps Engineer with hands-on experience in cloud infrastructure, automation, deployments, and production support. The ideal candidate should have experience with AWS, Kubernetes, CI/CD, and monitoring tools.

 Responsibilities

 - Manage and maintain cloud infrastructure (AWS/GCP/Azure).
- Deploy and support containerized applications using Docker and Kubernetes.
- Build and maintain CI/CD pipelines for automated deployments.
- Monitor systems, analyze logs, and troubleshoot production issues.
- Automate operational tasks using scripting and DevOps tools.
- Support incident management and root-cause analysis.

 Required Skills

 - 1+ year of DevOps/cloud infrastructure experience.
- Hands-on AWS experience (EC2, IAM, EKS, networking, storage, monitoring).
- Experience with Docker and Kubernetes.
- Experience with CI/CD tools (GitHub Actions, GitLab CI, Jenkins).
- Knowledge of monitoring tools (CloudWatch, Prometheus, Grafana, Datadog, ELK).
- Scripting knowledge in Bash/Python.
- Familiarity with Terraform/CloudFormation is a plus.
- Good understanding of Git and deployment workflows.

 Soft Skills

 - Strong troubleshooting ability.
- Ownership mindset and problem-solving approach.
- Good collaboration and communication skills.
